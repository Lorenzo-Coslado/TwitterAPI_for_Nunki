# Ouvrez deux nouveaux onglets dans votre navigateur

- http://127.0.0.1:5000/user=*****
- http://127.0.0.1:5000/search=*****/media=*****

- Remplacez les ***** par votre user_id, keyword, boolean
- WARNING : boolean ne peut être que "True" ou "False"
